# Sprint 2 Plan

## Sprint Goal

Expand NPC functionality and introduce campaign locations as a new core entity within the system.

This sprint enhances the usefulness of NPC records while establishing the foundational structure for tracking important places within a campaign world.

---

## Sprint Scope

Sprint 2 focuses on enriching NPC entries and introducing a second core campaign entity: locations.

The sprint prioritizes adding visual and descriptive depth to NPCs while implementing a full CRUD workflow for managing locations.

---

## Committed User Stories

- Add NPC Stat Block (5)
- Add NPC Image (5)
- Create Location (3)
- View Locations (2)
- Edit Locations (3)
- Delete Location (2)

**Total Commitment:** 20 Story Points

---

## Expected Outcome

By the end of Sprint 2 the application should support:

- Adding stat blocks to NPC entries
- Attaching images to NPC entries
- Creating location records
- Viewing a list of locations
- Editing existing locations
- Deleting locations

These capabilities expand LoreSmith’s campaign management tools while laying the groundwork for future relationship tracking between NPCs, factions, and locations.

---

## Notes

Sprint 2 builds on the NPC management functionality delivered in Sprint 1 while introducing locations as a new campaign entity. More advanced relationship tracking and visualization features will be deferred to future iterations.
